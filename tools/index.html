MathJax Testing Tools

These tools are currently for internal use, and have not been rigorously tested.

  • checklist.html
    Checks a cross-section of MathJax files highlighting success (or failure) and performance issues. This must be in the same domain as the MathJax installation.
  • performance.html
    Measures MathJax performance for a given page. Must be in the same domain as the page to be tested.
  • perf-analysis.html
    Generates pages of math and measures MathJax performance.
  • layout-comparison.html
    Simulates MathJax rendering to allow comparison of various layout options. Requires MathJax, but from any domain.
  • layout-demo.html
    Simulates MathJax rendering to allow comparison of various layout options. This test-page is a reduction of layout-comparison.html. Does not require MathJax.
  • layout-demo2.html
    Simulates MathJax rendering to allow comparison of various markup options. Does not require MathJax.
  • layout-demo3.html
    Tests performance of inserting pre-generated HTML representing math. This relates to the possible option of a low-quality mode (or phase) for MathJax. Does not require MathJax.

    • Finally, here is the code for a javascript bookmarklet which creates a diagnostics window for a web-page. This code requires the MathJaxToolsDir to be configured and to be in the same domain as the page under test.

    var MathJaxToolsDir = "/mathjax/tools"; var script = document.createElement("script"); script.src = MathJaxToolsDir + "/diagnostix.js"; document.getElementsByTagName("head")[0].appendChild(script); void(0);
